# 快速开始 - 3 步完成部署

## 在银河麒麟操作系统上部署图片查看器

### 步骤 1: 复制项目到麒麟系统

将整个 `TkinterSample` 文件夹复制到银河麒麟系统的任意位置。

### 步骤 2: 运行打包脚本

打开终端，进入项目目录，执行：

```bash
chmod +x build.sh
./build.sh
```

脚本会自动完成：

- ✓ 检查并安装依赖
- ✓ 打包应用
- ✓ 安装到用户目录
- ✓ 创建桌面快捷方式

### 步骤 3: 启动应用

桌面上会出现应用图标（名称可在配置文件中自定义），双击即可启动！

按 `ESC` 键退出。

---

## 自定义配置

### 修改应用名称和显示图片

编辑配置文件：

```bash
nano config.json
```

配置说明：

```json
{
  "image_path": "sample.jpg",
  "window_title": "图片查看器",
  "app_name": "我的应用"
}
```

- `image_path`: 要显示的图片路径（支持相对路径和绝对路径）
- `window_title`: 窗口标题
- `app_name`: **桌面图标显示的名称**（打包时使用）

**修改后重新运行 `./build.sh` 即可生效！**

### 已安装后修改配置

如果应用已经安装，编辑安装目录的配置文件：

```bash
nano ~/.local/share/applications/image-viewer/config.json
```

修改后重新运行打包脚本或手动更新桌面快捷方式：

```bash
./install-desktop-shortcut.sh
```

## 🚀 使用流程

### 在 Windows/Mac 上打包

1. **确保 Docker Desktop 正在运行**

2. **运行打包脚本**

   ```bash
   # Windows
   docker-build-windows.bat
   
   # Mac/Linux
   chmod +x docker-build-linux.sh
   ./docker-build-linux.sh
   ```

3. **获取结果**
   - 输出目录：`dist-linux/`
   - 可执行文件：`dist-linux/image-viewer`

4. **复制到麒麟系统并安装**

   ```bash
   cd dist-linux
   chmod +x *.sh
   sudo ./install-system.sh
   ```

## 💡 最简打包方案

如果只需要最基本的功能，只需以下 5 个文件：

2. `.dockerignore`
3. `docker-build-windows.bat` 或 `docker-build-linux.sh`

5. `requirements.txt`

其他文件（config.json、sample.jpg 等）可以在 Dockerfile 中动态添加。

## 🎯 快速验证

在有 Docker 的电脑上：

```bash
# 1. 检查 Docker
docker --version

# 2. 构建镜像
docker build -t kylin-packager .

# 3. 查看生成的文件
docker run --rm kylin-packager ls -lh dist/
```

如果看到 `dist/image-viewer`，说明打包成功！
